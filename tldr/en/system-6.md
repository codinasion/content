short_summary: "1988 Macintosh operating system"
description: "System 6 is a graphical User Interface based Operating System for macintosh Computers developed by Apple Computer. It was released in 1988 and is Part of the classic Mac Os Series. It is a monolithic operating system, with cooperative multitasking based on an improved MultiFinder. The boxed Version Cost us49 and was included with all new macintosh Computers until 1991 when it was followed by System 7."
