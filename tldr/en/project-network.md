short_summary: "Diagram showing the order of activities"
description: "A Network Diagram is a Graph that shows the Order in which Activities on a Project are to be completed. Derived from the Work Breakdown Structure, the terminal Elements of a Project are organized sequentially based on the Relationship among them. It is typically drawn from left to right to reflect the Chronology of the Project."
