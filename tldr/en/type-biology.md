title: "Type (biology)"
short_summary: "Specimen(s) to which a scientific name is formally attached"
description: "In Biology a Type is a particular Specimen of an Organism formally associated with the scientific Name of the Organism. In other Words a Type is an Example that serves to anchor or centralize the Defining Features of a Taxon. In older Usage the Type was a Taxon rather than a Specimen."
