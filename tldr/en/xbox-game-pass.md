title: "Xbox Game Pass"
short_summary: "Video game subscription service by Microsoft"
description: "Game Pass is a Subscription Service as Part of the Xbox and offered by microsoft Gaming. The Service launched on 1 June 2017 allows Users to download and play Video Games for xbox Video Games Consoles or microsoft Windows pcs from a rotating Library with the Games remaining available as long as the User has an active Subscription. Game Pass Subscribers also receive Discounts on Purchases of Games from the Service's library and their respective downloadable Content."
